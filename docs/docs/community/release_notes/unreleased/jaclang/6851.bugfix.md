- **Fix: `jac ai --ui` no longer fails looking for the removed `jac-client` package**: The web-UI launcher checked for an installed `jac_client` module and aborted with a now-impossible `pip install jac-client` hint. The client framework ships in `jaclang` core, so the check is gone and `jac ai --ui` works with no extra install.
