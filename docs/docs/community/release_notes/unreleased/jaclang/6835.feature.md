- **Feature: Single self-contained `jac` binary (linux-x86_64)**: `jaclang` can now be installed as a standalone native executable that statically embeds CPython and runs the core in-process, with no system Python, uv, or pip needed at install or runtime (`curl -fsSL .../scripts/install-binary.sh | bash`). The global cache root also falls back to a per-user temp dir when the preferred location (e.g. a read-only `$HOME`) is unwritable.
