- **Built-in MCP server with zero external dependencies**: The `jac mcp` Model Context Protocol server is now part of the `jac` binary instead of the separate `jac-mcp` PyPI plugin. The protocol (JSON-RPC 2.0 over stdio, streamable-HTTP, and SSE) is reimplemented on the Python standard library, so the server no longer pulls in the `mcp` SDK, pydantic, starlette, uvicorn, httpx, or cryptography. Run `jac mcp` directly with no install step.
