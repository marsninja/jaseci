- **CLI: Global npm CLI tools**: `jac install --npm --global <pkg>` now installs standalone npm CLI tools into a jac-owned global store (via the bundled bun, no project required), and `jac x --global <tool>` runs them — the npm analogue of `jac install --global` / `jac x --global` for Python. Tool locations for both `jac install` and `jac x` are unified behind a single `tool_paths` source of truth (scope × ecosystem), and `jac x`'s tier resolution is now an orthogonal `--global`/`--node` filter matrix.
