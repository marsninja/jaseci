- **jac-scale: `[dependencies.system]` for runtime OS packages**: Declare apt packages your app needs at runtime (e.g. `git`, `ffmpeg`) under `[dependencies.system]` in `jac.toml`. On a Kubernetes deploy they are installed into the **service container** at startup, so the binaries are present where the app actually runs -- fixing the binary-ship regression where system deps installed in the bootstrap init container never reached the running pod (#7115). Debian only; keys are apt package names.
