- **Refactor: Remove dead `categorize_symbol` helper**: `categorize_symbol` and the `SymbolCategory` enum were added speculatively to centralize scope classification across backends but were never used. They classify a symbol's definition location, whereas the backends actually track emission/binding state (native LLVM storage and the CPython `global` rebind rule; ecmascript's `let`-vs-reassign block tracking), so the helper fit no real need and has been removed.
