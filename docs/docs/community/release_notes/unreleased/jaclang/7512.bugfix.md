- **Fix: eight compiler/runtime bugs**: native sub-64-bit integer arithmetic (`i32`/`i16`/`i8`) no longer forces i64 overflow intrinsics; native `int / int` is true division (float) while `//` stays integer floor division; a `return` inside an `in <region> {}` open is counted as a guaranteed return (no spurious E1004); self-reassignment through a consuming call revives the owner (`total = combine(total, step)` in a loop no longer reports use-after-move); a backtick-`root` ability trigger is rejected at check time (E1116, pointing to `Root`) instead of crashing at dispatch; typed disconnect (`a del ->:E:-> b`) emits the correct `filter=`/`dir=` runtime keywords; `jac start --no-client` accepts the documented hyphenated spelling; and `console.error` no longer prints a doubled `Error: Error:` banner for pre-prefixed runtime tracebacks.
