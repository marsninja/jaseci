- **Native: generators, closures, static decorators, mixed isinstance, and os.path complete the zero-global-cost gap sweep**: `yield` functions declared `-> Iterator[T]` now compile natively as resumable JacIter state machines (for-loops, first-class `next()`, and the lazy adapters consume them unchanged, and `map`/`filter` accept inline non-capturing lambdas); nested functions may capture enclosing locals (reads late-bound, rebound captures write through to the enclosing frame, escape shapes are structured errors); module-level functions accept static name decorators applied at glob-init through a real call slot (registration and redirection both work), while method/archetype/dynamic decorators are now rejected instead of silently dropped; `isinstance` over tuples mixing builtins and archetypes lowers as a guarded tag/vtable disjunction with subtype matching; and the bundled native `os.path` gains `join` and `abspath`. Every lowering costs only its use sites; object layout and shared codegen paths are untouched for code not using these features.
