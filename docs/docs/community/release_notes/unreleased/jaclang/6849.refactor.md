- **Refactor: enhanced console and shadcn/ui now ship with jaclang core**: The Rich-style console output (colors, box-drawn panels and tables, animated spinners) is reimplemented in pure Jac and is now the default, so jaclang stays free of the `rich` dependency. The shadcn/ui system (`jac add --shadcn`, `jac create --use jac-shadcn`, `jac retheme`) is now a built-in provider. The separate `jac-super` package is removed; nothing needs to be installed for either feature.
