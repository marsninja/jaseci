- **Fix: ANSI color codes no longer leak into redirected/non-TTY streams**: The base ANSI `ConsoleRenderer` now checks `isatty()` on the *target* stream at emit time, not just the global `use_color` flag resolved from `sys.stdout` at startup. `jac build 2>errors.log` with an interactive terminal no longer writes raw escape codes into `errors.log`, and `console.print(..., file=sink)` stays plain text for any non-TTY sink.
