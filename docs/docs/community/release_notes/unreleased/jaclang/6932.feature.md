- **Native `isinstance()` over user-defined types**: The native backend now lowers `isinstance(x, MyClass)` and `isinstance(x, (A, B))` via an inheritance-aware vtable-identity check (matching the target class and all its subclasses, correct under multiple inheritance), and narrows field access inside the guard so `x.field` resolves on the narrowed type. Previously these raised a "native pathway does not yet support" error (E5090).
