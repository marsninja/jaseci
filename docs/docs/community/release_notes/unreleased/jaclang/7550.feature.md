- **CLI: `jac guide` now bundles the full documentation set**: The agent-relevant sections of the docs site ship inside the `jac` CLI as searchable doc sets -- `reference/`, `quick-guide/`, `tutorials/`, `internals/`, and `community/` (breaking changes and the codebase guide) -- served from the same files the documentation site is built from. `jac guide <set>/<name>` prints any doc (a unique bare name also resolves), `jac guide <set>` lists a set, `jac guide --search "<terms>"` greps every bundled guide and doc with `name:line:` hits, and `jac guide --json` emits the full machine-readable index. The MCP server exposes the same corpus as `jac://docs/*` resources.
