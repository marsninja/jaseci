- **Fix: `jac install` no longer destroys `jac.toml` comments or declared version constraints**: installing a package rewrote the whole manifest from a parsed dict, deleting every comment, and resolved past explicitly declared bounds, silently rewriting pins like `bcrypt = ">=4.0.0,<5.0.0"` to `~=5.0`. Manifest writes are now surgical text edits that preserve comments, formatting, and key order, and a bare `jac install <pkg>` resolves within the declared constraint and leaves the pin untouched; an explicit spec on the command line still wins, but replacing a declared constraint is announced in the output.
