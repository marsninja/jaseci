- **Fat jab (`jac build --fat`)**: The jab and binary projections can now vendor the app's runtime Python dependency closure as wheels into the bundle (under `_vendor/wheels/`, a sibling of `_precompiled/` that leaves the sealed manifest untouched). At materialize time the wheels install offline into a cache-scoped site and go on `sys.path`, so a fat jab runs with no PyPI access; the wheels ride inside the tarball, so the existing sha256 content addressing covers them for free.
