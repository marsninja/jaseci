- **ownbench: gradual-ownership benchmark suite (`jac/examples/ownbench`)**: six kernels (binarytrees, vecdot, histogram, vm, rbtree, deriv), each a single fully-annotated source compiled under `--enforce-nogc --gc none --assert-no-rc`, `--gc rc`, and `--gc cycles`, with byte-identical output across modes as the differential oracle. Pure-Jac harness (`jac run harness/measure.jac|audit.jac|erase.jac`) for timing/RSS measurement, IR-level `__rc_*` audits (enforced builds are zero), and annotation erasure that must reproduce the annotated digest; `ci_identity.sh`/`run_all.sh` entry points, plus `test_ownbench_differential.jac` wiring the kernels into the test suite as the regression net for headerless codegen.
