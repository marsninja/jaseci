- **Bundled interpreter switched to free-threaded CPython (3.14t)**: The `jac` binary now ships and `dlopen`s the free-threaded (`--disable-gil`) python-build-standalone CPython (`libpython3.14t`), staging the `python3.14t` stdlib alongside the `python3.14t` interpreter. The build downloads the `freethreaded+pgo+lto-full` pbs flavor. `setuptools` and `wheel` are now bundled into the binary's site so PEP 517 sdist/editable builds keep working on the free-threaded ABI where no `cp314t` wheel exists yet. CI no longer provisions any system Python (`actions/setup-python` is gone from every workflow); all Python tooling now runs through the binary via `jac -m`.
