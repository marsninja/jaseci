- **Fix: Resolve project-root-absolute imports through namespace packages during `jac test`**: A test suite whose project-root-absolute import (e.g. `import from engine.math.vec3 { ... }`) descended through an *intermediate* namespace package (one holding only subpackages and no direct `.jac` file) failed to import during pytest collection with `ModuleNotFoundError`, so a configured `[test] directory` silently collected 0 tests. `JacMetaImporter` (and the lazy bootstrap finder) now recognize a directory as an implicit Jac namespace package when a `.jac` source lives anywhere in its subtree, matching how the native runner resolves the same import.
