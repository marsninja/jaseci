- **Breaking: `own` and `mut` are now reserved keywords (#7398)**: The ownership qualifiers introduced in #7149 previously parsed as contextual names, so any identifier could appear in an ownership-prefix position. `own` and `mut` are now reserved tokens (`KW_OWN`/`KW_MUT`), pinning the surface before more work stacks on it -- code that used `own` or `mut` as a variable, parameter, or field name must rename it (`val` stays contextual and is unaffected). This is a source-level break only; the ownership semantics are unchanged.
