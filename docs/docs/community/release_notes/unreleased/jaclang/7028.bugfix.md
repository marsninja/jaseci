- **Fix: keyword arguments dropped in native calls**: On the native (`na`) backend, keyword arguments in a function or method call were silently dropped: the call fell back to the parameters' defaults instead of the values passed by keyword (so `greet(name="abcd", n=7)` ran the all-defaults body). Calls now bind keyword arguments to their named parameters (reordering as needed) across same-module, method, and cross-module (`.na.jac`) calls, matching the Python pathway.
