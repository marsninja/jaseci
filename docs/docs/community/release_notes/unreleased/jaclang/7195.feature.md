- **`jac eject` emits a runnable scaffold**: The ejected app now carries the project's Python `[dependencies]` into `backend/requirements.txt`, copies `.env`/`.env.example` and loads them at startup, and ships a top-level `Makefile` (`install`/`dev`/`build`/`start`) plus a README quickstart, so `make install && make dev` runs the app without hand-wiring.
