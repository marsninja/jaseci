- **Native (#6951): static methods + class attributes, and a bundled `datetime`**: The native (na) pathway now lowers static-method calls (`ClassName.method(args)` on a `static def`) and class-attribute reads (`ClassName.attr` on a `static has`) on user classes, resolved through the MRO so subclasses inherit statics, and working both same-module and cross-module (a `Class.method(...)` call on an imported class now pulls the class's defining module into the native link set). Riding this, a new bundled `na_stdlib/datetime.na.jac` ships a UTC `datetime`/`timezone` pair where the same `import from datetime { datetime, timezone }` binds CPython `datetime` on the sv pathway and the native module on na: `timezone.utc` is a class attribute and `datetime.now`/`datetime.fromtimestamp` are class-level constructors, with the civil date computed from the POSIX epoch (Hinnant's days->civil) so `year`/`month`/`day`/`hour`/`minute`/`second`, `weekday()`, and `isoformat()` match CPython exactly for a fixed timestamp. Scope is UTC / fixed-offset only (no tz database, DST, leap seconds, or microseconds).
