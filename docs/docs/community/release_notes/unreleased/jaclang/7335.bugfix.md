- **Fix: jac-scale topology index (GTI) is now invalidated cross-pod on edge-list-delta writes**: In a multi-pod deployment backed by `ScaleTieredMemory` (Mongo L3 + Redis L2), adding or removing an edge on a root refreshed the root's topology index in Mongo but left a stale copy in Redis and published no cross-pod invalidation, so other pods kept serving a stale GTI and type-narrowed traversals like `[root ->[?:Type]]` returned stale results on any pod that did not perform the write. `commit` now invalidates L2 and broadcasts an invalidation for edge-list-delta writes whose anchor carries a topology index (a root), so other pods evict their L1 and reload the fresh index from L3. Storm-avoidance is preserved for ordinary content nodes, and the behavior is unchanged when the topology index is disabled.
