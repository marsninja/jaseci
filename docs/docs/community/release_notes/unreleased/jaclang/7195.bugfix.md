- **`jac eject` output now boots, serves, and uploads like `jac start`**: Endpoints are harvested only from the entry module's import closure (standalone `with entry` scripts no longer hijack uvicorn at boot), `@restspec(path=...)` routes are mounted at their declared paths, the client runtime's sibling chunks and compiled `pages/` router are copied so the frontend renders, and the dev proxy targets `127.0.0.1` instead of `localhost` to avoid IPv6 `ETIMEDOUT` stalls. Functions taking an `UploadFile` now generate a multipart `File()` endpoint (fixing the `UnicodeDecodeError` crash on file upload) and pull in `python-multipart`. Resource files are split cleanly (web assets to the frontend, other data to the backend), docstring-only modules are no longer emitted as backend modules, and a prior eject output is excluded from a re-eject to a different directory.
