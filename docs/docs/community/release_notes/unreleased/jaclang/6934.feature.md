- **Native: partial-native classes / per-method opt-out**: A native-eligible class may now contain methods the native backend cannot lower (e.g. one using `eval`) without poisoning its whole module's native compile. Such a method is demoted to its Python implementation while the class keeps its native layout, native constructor, and lowerable sibling methods; construction-critical `init`/`postinit` gaps remain hard errors. Each demotion prints a terminal warning naming the method so a silent native fallback is visible during a build; set `warn_native_seams = false` under `[check]` in `jac.toml` for quiet builds (`JAC_NATIVE_DEBUG=1` forces the warning on regardless).
