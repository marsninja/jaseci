- **Native: upgrade to LLVM 22 and range-fetch LLVM via slices**: The native (`na`) backend now builds on LLVM 22.1.8 (the `LLVMPY_*` shim re-vendored verbatim from llvmlite 0.48.0rc1). The Linux shim links against GNU libstdc++ to match the official LLVM 22 Linux release (statically, so the shipped shim stays self-contained); macOS continues via Apple `clang++`/libc++. `zig build fetch-llvm` now range-fetches only the ~84 MB the shim links (`lib/libLLVM*.a` + `llvm`/`llvm-c` headers, plus macOS `libLTO.dylib`) from the `jaseci-labs/llvm-slice` repackaged zip instead of the ~1 GB upstream `.tar.xz`, with linked archives verified against a pinned manifest checksum.
