- **`[dev] test_jobs` configures the test runner's worker count**: `jac test` now reads the default xdist worker count from `[dev] test_jobs` in `jac.toml` (`"auto"` = one worker per core, `"0"` = serial, or a fixed count like `"4"`), so you no longer have to prefix every run with `JAC_TEST_JOBS=`. The `JAC_TEST_JOBS` environment variable still overrides the config when set.
