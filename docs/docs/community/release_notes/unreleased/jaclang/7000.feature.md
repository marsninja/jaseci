- **Feature: opt-in deslop lint rules + `jac precommit`**: Adds two destructive, opt-in auto-lint rules, `strip-comments` (W3050) and `strip-docstrings` (W3051), that remove all comments/docstrings only when named explicitly in `[check.lint]` select (never under `all`/`default`), plus a `jac precommit` command that runs `jac format --lintfix` and `jac check` using `jac.toml` settings and can install a git pre-commit hook.
