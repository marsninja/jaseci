- **Feature: Single self-contained `jac` binary**: `jaclang` now ships as one native executable (a Zig launcher that `dlopen`s a bundled CPython and runs the core in-process) built with a single `zig build`. No system Python, uv, or pip is required at install or runtime, and `jac test` runs the suite through the binary. The binary is the distribution -- jaclang is no longer published to PyPI; plugins consume jaclang as a host dependency.
