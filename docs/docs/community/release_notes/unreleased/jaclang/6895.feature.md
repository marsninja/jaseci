- **Feature: native `jac` binary released for all four platforms**: The release pipeline now builds and attaches the self-contained `jac` binary for `linux-x86_64`, `linux-aarch64`, `macos-x86_64`, and `macos-aarch64` (one native runner each), matching every platform the installer detects. The obsolete pex/scie standalone build (which pip-installed `jaseci` from PyPI and could no longer resolve `jaclang`) was removed; the native binary built by the release-publish workflow is the sole distribution. `scripts/install.sh` downloads this binary, and the installer CI now tests that download/run/uninstall path.
