- **Breaking: the `jaseci` meta-package is retired; releases are anchored on jaclang**: `pip install jaseci` is no longer published. jaclang ships as the native `jac` binary and is now the release anchor -- a release that bumps jaclang cuts the `v<version>` GitHub Release (marked latest) that carries the binaries, while plugins (byllm/jac-scale/jac-mcp) publish to PyPI independently. Install Jac with the one-line installer and add plugins via `jac install <plugin>`.
