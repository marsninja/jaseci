- **Feature: GDB-debuggable native binaries (`jac nacompile -g`)**: Passing `-g`/`--debug` to `jac nacompile` emits DWARF line tables and `DISubprogram`s and has the pure-Python ELF linker relocate the `.debug_*` sections and write a `.symtab` plus a full section header table, so `gdb`/`lldb` can resolve source-level breakpoints and backtraces. `-g` implies `--scrub` so a cached non-debug build isn't reused; non-debug builds remain byte-identical to before.
