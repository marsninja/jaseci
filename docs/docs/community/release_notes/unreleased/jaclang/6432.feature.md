- **Native: structural `match`, nested closures, and `isinstance`/`type`**: The `.na.jac` native (LLVM) backend now lowers structural match patterns (sequence, `*`-star, mapping, and class/archetype patterns with positional and keyword sub-patterns), nested `def`s that capture enclosing-function locals (including recursion), `isinstance(x, T)` (inheritance-aware for archetypes, static for primitives), and `type(x)` (returns the value's type name as a string). Builtins that can never exist in a native binary (`eval`, `exec`, `compile`, `globals`, `locals`, `vars`, `dir`, `getattr`, `setattr`, `hasattr`, `delattr`, `__import__`) now fail loudly with the new `E5091` diagnostic instead of being silently dropped. (jaseci-labs/jaseci#6432)
