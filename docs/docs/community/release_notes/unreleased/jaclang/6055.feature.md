- **`pending` clause on `try`**: A new contextual clause -- `try { ... } pending { ... }` -- names the JSX (or value) to use during the dispatched-but-not-joined window of any async work inside the `try` body. On the `cl` target the view-lowering pass wraps the slot in `<JacPending fallback={...}>{...}</JacPending>` from `@jac/runtime` (a `React.Suspense` shim); on `sv` and `na` the pending body is dropped with a new `W2020` warning until the streaming-SSR and native-thread lowerings land. `finally` alongside `pending` is rejected with `E2022`.
