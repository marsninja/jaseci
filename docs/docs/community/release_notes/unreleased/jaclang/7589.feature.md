- **Feature: native is the default codespace**: markerless `.jac` modules with no server-requiring constructs now compile whole-module native and execute through the native engine by default -- no flags, no config. `[build] default_codespace = "sv"` opts a project back to the Python backend, and behavior is identical either way (the native stdlib is congruent with CPython by contract; inferred modules that fail to lower demote to the server codespace with a dim `note:`). Server-shaped code (OSP archetypes, python imports, serve endpoints, JSX, test blocks, match patterns) compiles exactly as before. Boundary rules shipped with the flip: analysis-only compiles (`jac check`, language server) keep stable server diagnostics; python importers of an inferred-native module always get its plain python side (native execution belongs to `jac run`, compiled binaries, and native importers); and the serve runtime pins the server codespace. Existing projects need no changes -- pure modules simply get native speed, and more code migrates automatically as native lowering coverage grows.
