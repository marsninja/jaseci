- **Feature: `jac ai --tui` Ink terminal UI**: Adds an interactive Ink/React TUI for the Jac coding agent via a jac-super plugin that compiles `runtime.cl.jac` on demand, launches a lightweight agent HTTP server, and streams responses over SSE (same bridge as `jac ai --ui`). Requires Node.js 22+ and jac-super installed. (jaseci-labs/jaseci#6355)
