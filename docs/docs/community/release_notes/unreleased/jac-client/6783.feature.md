- **Client-only (wasm) builds**: A project with `kind = "wasm"` is now auto-detected by `jac build` and `jac start` (no `--client wasm` flag needed). `jac build` inlines the JS bundle and CSS into a self-contained `index.html` that opens directly from disk (`file://`), and `jac start` serves it with a minimal static server (no API server, auth, or database) that still maps `/static/<name>.wasm` for `na`->wasm modules. An explicit `--client` overrides the auto-detection.
